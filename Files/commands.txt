General:-> F1 > Command Palette-> strg + p > Open Files-> strg + num- > close window-> strg + num+ > new window-> strg + , > settingsCode:-> ALT + down/up > up down line-> Shift + Alt + down/up > copy line -> STRG + Shift + K > delete line-> STRG + Shift + Enter > add line above-> CTRL + ] /[ > tab line-> Home or POS1 > start end line-> STRG Home or POS1 > start end fileComments:-> Shift + Alt + A > '''asd'''-> STRG + 2 > "-> Ctrl + # > #-> Ctrl + K Ctrl + C > Add line comment-> Ctrl + K Ctrl + U > Remove line comment
